# День рождения клуба Ульяны Мамедовой · 1 год

Мобильное веб-приложение для гостей праздника 11 июля 2026 года.
Пять экранов: главная, программа дня, девять зон, партнёры и карта площадки.

## Как выложить на GitHub Pages

1. Создайте новый репозиторий на GitHub (например, `club-1-year`).
2. Загрузите в него **всё содержимое этой папки** — файл `index.html` должен лежать в корне репозитория, а не внутри вложенной папки.
3. Откройте **Settings → Pages**.
4. В разделе *Build and deployment* выберите:
   - Source: **Deploy from a branch**
   - Branch: **main**, папка **/ (root)**
5. Нажмите **Save** и подождите 1–2 минуты.

Сайт будет доступен по адресу `https://ВАШ_ЛОГИН.github.io/club-1-year/`.

Гостья открывает ссылку с телефона и может добавить приложение на домашний экран —
оно откроется на весь экран, без адресной строки браузера, как обычное приложение.

## Как посмотреть локально

Откройте `index.html` двойным кликом — всё работает без сервера.
(Оффлайн-режим включается только на настоящем сайте.)

## Что внутри

```
index.html              всё приложение: разметка, стили и логика в одном файле
sw.js                   оффлайн-режим
manifest.webmanifest    чтобы приложение ставилось на домашний экран
.nojekyll               отключает обработку Jekyll на GitHub Pages
assets/img/
  hero.jpg              фото на главную
  map.jpg               карта площадки
  icon-*.png            иконки приложения
  partners/<партнёр>/   1.jpg … N.jpg — фотографии, logo.png — логотип
```

## Как что-то поменять

Все тексты, время в программе, ссылки на соцсети и списки фотографий лежат
в одном месте — внутри `index.html`, в блоке `<script>` после комментария
`ДАННЫЕ — всё взято из материалов организаторов`. Там три структуры:

- `PARTNERS` — карточки партнёров: имя, описание, адрес, соцсети,
  `photos` — сколько фотографий лежит в папке партнёра;
- `ZONES` — девять активностей на площадке;
- `PROGRAM` — расписание дня.

Чтобы добавить фотографию партнёру: положите её в его папку под следующим
номером (например, `6.jpg`) и увеличьте `photos` на единицу.

Чтобы поменять дату: исправьте `EVENT_START` и `DAY` в начале блока данных —
от них считаются обратный отсчёт и отметка «Сейчас» в программе.

Цвета собраны в самом верху файла, в блоке `:root` — например, `--gold`
и `--bg`. Поменяв их там, вы поменяете цвет во всём приложении.

**Важно:** если вы обновили файлы на сайте, поднимите номер версии
в `sw.js` (`const VERSION = "v2"` → `"v3"`), иначе у гостей, которые уже
заходили, останется старая сохранённая копия.

## Что умеет

- обратный отсчёт до 14:00 11 июля, а в день праздника — плашка «Идёт сейчас»
  и золотая точка напротив текущего пункта программы;
- избранные зоны отмечаются сердечком и сохраняются в телефоне гостьи;
- кнопка «назад» на телефоне закрывает окно, а не выбрасывает из приложения;
- карта открывается на весь экран с увеличением по нажатию;
- фотогалереи партнёров листаются свайпом;
- открывается без интернета после первого захода.
